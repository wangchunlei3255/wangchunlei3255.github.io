基于github和hexo的个人网站的搭建

  1. 下载安装nodejs。
    win+r打开cmd，输入node -v ，显示版本即为安装成功
  2. 注册GitHub
    进入GitHub官网注册账号
  3. 创建仓库，格式为：用户名.github.io（就这么写，我也不知道为啥，还没研究明白。）

  4. 在仓库中新建一个index.html文件作为初始页面
  5. 下载安装git，慢的话去别的网站下载，安装过程去百度
  6. 配置githubSSH
    在安装路径下打开git-bash.exe

    输入指令：ssh-keygen -t -C “GitHub注册时的邮箱“ 直接回车
    打开GitHub——头像——设置——打开ssh，在C盘——用户.ssh文件夹下，用记事本打开id_rsa.pub，把内容复制到前一步的SSHkey中
  7. 安装hexo框架
    在选好的路径下win+r打开cmd，输入hexo安装命令
  8. 初始化
    输入指令：hexo init 你的博客名，回车
  9. 初始化结束后，进入目录，安装：npm install
    安装好后可以本地预览：hexo s -p 5555
    浏览器输入localhost：5555
  10. 编辑器可以使用vscode
  11. 安装hexo-deployer-git插件，指令：npm install hexo-deployer-git –save
  12. hexo发布的GitHub上需要进行的配置
    deploy:
    type: git
    repo: 地址为GitHub的地址
    branch: master
  13. 在博客文件夹下右键打开git bash here，配置邮箱和用户名
  14. 监视文件变动：hexo g
    部署 ：hexo d
    组合：hexo g -d
